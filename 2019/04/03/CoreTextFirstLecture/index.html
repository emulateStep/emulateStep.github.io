CoreText第一讲 常用类和方法

发表于 2019-04-03 | 分类于 CoreText

CoreText第一讲

CoreText常用到的类和方法

常用类

  • CTFontRef
  • CTRunDelegateCallbacks
  • CTRunDelegateRef
  • CFAttributedStringRef
  • CTFramesetterRef
  • CGPathRef
  • CTFrameRef
  • CTLineRef
  • CTRunRef
  • CFArrayRef
  • CFIndex
  • CFRange

常用方法（按照项目中方法使用顺序排序）

  1. 创建CTRunDelegateRef, 需要__bridge_retained桥转，防止被系统回收
    CTRunDelegateRef _Nullable CTRunDelegateCreate( const CTRunDelegateCallbacks* callbacks, void * _Nullable refCon ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  2. 设置RunDelegate代理
    range : CFRangeMake(0, 1),CTRun为图片，起始0，长度1
    attrName ：kCTRunDelegateAttributeName
    CF_EXPORT void CFAttributedStringSetAttribute(CFMutableAttributedStringRef aStr, CFRange range, CFStringRef attrName, CFTypeRef value);

  3. 创建CTFontRef字体
    CTFontRef CTFontCreateWithName( CFStringRef name, CGFloat size, const CGAffineTransform * _Nullable matrix ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  4. 创建CTFramesetterRef
    CTFramesetterRef CTFramesetterCreateWithAttributedString( CFAttributedStringRef string ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  5. 创建CTPathRef
    CG_EXTERN CGPathRef CGPathCreateWithRect(CGRect rect, const CGAffineTransform * __nullable transform) CG_AVAILABLE_STARTING(10.5, 4.0);

  6. 创建CTFrameRef
    framesetter : CTFramesetterRef
    stringRange : 设置多大就显示多少字符。设置length为0时，一直渲染，直到渲染的字符串结束或者超出path设置的范围
    frameAttributes ：额外控制frame的属性字典，没有时填入NULL
    CTFrameRef CTFramesetterCreateFrame( CTFramesetterRef framesetter, CFRange stringRange, CGPathRef path, CFDictionaryRef _Nullable frameAttributes ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  7. frame的范围转换成此frame在全部字符串中的Range
    CFRange CTFrameGetVisibleStringRange( CTFrameRef frame ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  8. 获取组成frame的所有CTLine数组
    CFArrayRef CTFrameGetLines( CTFrameRef frame ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  9. 获取每一行的起始点，保存在origins数组中
    frame : 存储有所有行信息，传入的frame的所有行坐标点会被复制一份
    range : 想要获取的行坐标的范围，如果range的length是0，复制操作会重新从range的start位置继续，直到复制完成
    origins : 一个CGPoint类型的数组指针，用来接收从frame中复制出来的行坐标 void CTFrameGetLineOrigins( CTFrameRef frame, CFRange range, CGPoint origins[_Nonnull] ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  10. 获取CTLine包含的所有的CTRun
    CFArrayRef CTLineGetGlyphRuns( CTLineRef line ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  11. 获取CTRun保存的属性。获取到的内容是通过CFAttributedStringSetAttribute方法设置给图片属性字符串的NSDictionary，key为kCTRunDelegateAttributeName，值为CTRunDelegateRef
    CFDictionaryRef CTRunGetAttributes( CTRunRef run ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  12. 从delegate中获取使用CTRunDelegateCreate初始时候设置的元数据
    void * CTRunDelegateGetRefCon( CTRunDelegateRef runDelegate ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  13. 获取CTRun的绘制属性ascent、desent，返回值是CTRun的宽度
    double CTRunGetTypographicBounds( CTRunRef run, CFRange range, CGFloat * _Nullable ascent, CGFloat * _Nullable descent, CGFloat * _Nullable leading ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  14. 获取CTRun在整段文字中的Range
    CFRange CTRunGetStringRange( CTRunRef run ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  15. 获取整段文字中charIndex位置的字符相对所在line的原点的x值
    CGFloat CTLineGetOffsetForStringIndex( CTLineRef line, CFIndex charIndex, CGFloat * _Nullable secondaryOffset ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  16. 获得CTFrame的绘制区域
    CGPathRef CTFrameGetPath( CTFrameRef frame ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  17. 获取边界框
    CG_EXTERN CGRect CGPathGetBoundingBox(CGPathRef cg_nullable path) CG_AVAILABLE_STARTING(10.2, 2.0);

  18. rect 按照（dx，dy）进行平移
    CG_EXTERN CGRect CGRectOffset(CGRect rect, CGFloat dx, CGFloat dy) CG_AVAILABLE_STARTING(10.0, 2.0);

  19. 获取数组中的元素个数
    CF_EXPORT CFIndex CFArrayGetCount(CFArrayRef theArray);

  20. 这是一个实现相对位移的函数，每次位移都是相对于UIKit坐标下的（0，0）点
    CG_EXTERN CGAffineTransform CGAffineTransformMakeTranslation(CGFloat tx, CGFloat ty) CG_AVAILABLE_STARTING(10.0, 2.0);

  21. 一个用于处理形变的类,其可以改变控件的平移、缩放、旋转等,其坐标系统采用的是二维坐标系,即向右为x轴正方向,向下为y轴正方向
    t ： CGAffineTransform
    sx ： 当sx为正值时，会在x轴方向上缩放x倍，反之，则在缩放的基础上沿着竖直线翻转
    sy ： 当sy为正值时，会在y轴方向上缩放y倍，反之，则在缩放的基础上沿着水平线翻转
    CG_EXTERN CGAffineTransform CGAffineTransformScale(CGAffineTransform t, CGFloat sx, CGFloat sy) CG_AVAILABLE_STARTING(10.0, 2.0);

  22. 获取CTLine数组中第idx个line元素，
    CF_EXPORT const void *CFArrayGetValueAtIndex(CFArrayRef theArray, CFIndex idx);

  23. 获取CTLine的上行高度，下行高度，返回值是CTLine的行距
    double CTLineGetTypographicBounds( CTLineRef line, CGFloat * _Nullable ascent, CGFloat * _Nullable descent, CGFloat * _Nullable leading ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  24. 坐标系转换
    CG_EXTERN CGRect CGRectApplyAffineTransform(CGRect rect, CGAffineTransform t) CG_AVAILABLE_STARTING(10.4, 2.0);

  25. 获取点击处position文字在整段文字中的index
    CFIndex CTLineGetStringIndexForPosition( CTLineRef line, CGPoint position ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));

  26. 获取CTLine中文字在整段文字中的Range
    CFRange CTLineGetStringRange( CTLineRef line ) CT_AVAILABLE(macos(10.5), ios(3.2), watchos(2.0), tvos(9.0));
